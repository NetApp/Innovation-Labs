NetApp Connector (Project Neo) Documentation ​

Welcome to the NetApp Connector (Project Neo) documentation. This guide provides comprehensive information about deploying, managing, and troubleshooting the NetApp Connector (Project Neo).

❗ Important: Project Neo for M365 Copilot is currently in Private Preview. This means that the connector is not yet fully supported and may have some limitations. The connector requires a license to activate. You can request access to the connector by joining the Early Access Program (EAP).
